# Helix Bank · Next.js app

This subfolder contains the Next.js 16 application. For the full project README — including the scenario, the 4 LD artefacts walkthrough, setup, architecture, and decision-log pointer — see the repo root: **[../README.md](../README.md)**.

## Quick start

```bash
npm install
cp ../.env.example .env.local  # then edit with real values
npm run dev
```

Visit `http://localhost:3000` (or whichever port Next picks).

See `../README.md` for the full setup, including which env vars are required and where to source them.
